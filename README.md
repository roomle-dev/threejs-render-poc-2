# three.js render PoC 2

## Install and build

```lang-none
npm i  
npm run dev
```

```lang-none
npm i  
npm run build
npm run serve
```

## URL Parameters

| name | type | description | example |
|------|------|-------------|---------|
| `type` | `string` | Renderer type 'webgl', 'webgpu-forcewebgl', 'webgpu' (default 'webgl'). | `type=webgpu` |
